# QCM Révisions

Petit projet web de révision sous forme de QCM.

## Description

Ce site permet de réviser deux cours :

- Développement Web
- Culture de la Création

L’utilisateur peut lire le cours, lancer un QCM, répondre aux questions et voir son score à la fin.

## Fonctionnalités

- Choix entre deux QCM
- Accès au cours complet
- Questions mélangées à chaque tentative
- Réponses mélangées
- Affichage du score final
- Historique des essais
- Possibilité d’imprimer ou sauvegarder le cours en PDF

## Technologies utilisées

- HTML
- CSS
- JavaScript

## Comment utiliser le projet

1. Télécharger ou cloner le projet.
2. Ouvrir le fichier `index.html` dans un navigateur.
3. Choisir un cours ou lancer directement un QCM.
